## 📂 Document Categorization

Document Categorization is a key feature of the Intelligent Document Management System. It helps users organize uploaded documents into suitable categories for easy searching, filtering, and management.

### How It Works

1. User uploads a document.
2. The user selects a document category.
3. The document is stored under the selected category.
4. Users can search and filter documents based on categories.

### Categories

- **Academic Documents**
  - Mark Sheets
  - Certificates
  - Resume

- **Financial Documents**
  - Invoices
  - Bills
  - Receipts

- **Office Documents**
  - Reports
  - Offer Letters
  - Salary Slips

- **Legal Documents**
  - Agreements
  - Contracts

- **Identity Documents**
  - Aadhaar Card
  - PAN Card
  - Passport

- **Medical Documents**
  - Prescriptions
  - Medical Reports

- **Other Documents**
  - Miscellaneous Files

### Benefits

- Easy document organization
- Faster document search
- Simple category-based filtering
- Better document management
