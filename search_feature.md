# Search Feature

The Keyword Search feature helps users find documents quickly without opening every file manually.

## Search Options

Users can search documents using:

- File name
- Document category
- Tags
- OCR extracted text
- File type
- Upload date

## How It Works

1. The user enters a keyword in the search box.
2. The system checks document names, categories, tags, and extracted text.
3. Matching documents are displayed to the user.
4. The user can preview, download, or manage the selected document.

## Benefits

- Saves time while finding documents
- Makes scanned documents searchable
- Reduces manual document searching
- Improves document retrieval and management
